Создайте страницу, на которой будут размещены теги логической разметки. Запустите этот и предыдущий примеры в разных браузерах. Отличаются ли элементы в разных браузерах?

Логическое форматирование текста – это форматирование текста предназначенное для удобства работы с ним поисковых систем и прочих программ, например, различных читалок. А вот и теги которые применяются для логической разметки:

< abbr > ... < /abbr > - указывает на то что текст внутри тега является аббревеатурой. С помощью атрибута title дается расшифровка сокращения, что позволяет понимать аббревиатуру тем людям, которые с ней не знакомы. Кроме того, поисковые системы индексируют полнотекстовый вариант сокращения, что может использоваться для повышения рейтинга документа.

< cite > ... < /cite > - делает текст сноской или цитатой.

< code > ... < /code > - предназначен для отображения одной или нескольких строк текста, который представляет собой программный код. Сюда относятся имена переменных, ключевые слова, тексты функции и т.д. Как правило тображается моноширным текстом меньшего размера.

< strong > ... < /strong > - тег является тегом логической разметки и определяет важность помеченного текста (очень похож на тег < b > но это не одно и тоже). Задает тексту жирное начертание.

< em > ... < /em > - указывает браузеру на важность текста и задает ему курсивное начертание.

< del > ... < /del > - используется для выделения текста, который был удален в новой версии документа.

< ins > ... < /ins > - предназначен для выделения текста, который был добавлен в новую версию документа.

< mark > ... < /mark > - омечает текст как выделенный. Такой текст ничем не отличается от обычного, но его вид может быть изменен с помощью стилей. В браузере Chrome и Firefox фоновый цвет текста внутри выделяется желтым цветом.

< q > ... < /q > - выделяет текст в цитату.Содержимое контейнера автоматически отображается в браузере в кавычках.

< samp > ... < / samp > - используется для отображения текста, который является результатом вывода компьютерной программы или скрипта. Браузеры обычно отображают текст с помощью моноширинного шрифта.

< var > ... < / var > - используется для выделения переменных компьютерных программ. Браузеры обычно помечают текст курсивным начертанием.
